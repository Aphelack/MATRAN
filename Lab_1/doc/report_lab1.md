### 2 ОПРЕДЕЛЕНИЕ ПОДМНОЖЕСТВА ЯЗЫКА ПРОГРАММИРОВАНИЯ SWIFT

Цель подмножества — зафиксировать минимально достаточные конструкции для дальнейшей реализации интерпретатора Swift на F#. Подмножество охватывает базовые типы, управляющие конструкции и структуры данных, необходимые для учебных задач и демонстрации семантики языка.

**Литералы и константы.** В подмножество включены:
- целочисленные литералы: `0`, `42`, `-7`;
- литералы с плавающей точкой: `3.14`, `2.0e2`;
- строковые литералы: `"Hello"`;
- символьные литералы: `"A"` (тип `Character`);
- булевы литералы: `true`, `false`.

**Типы данных и переменные.** Используются базовые типы Swift:
- `Int`, `Double`, `Bool`, `String`, `Character`;
- переменные объявляются через `var`, константы через `let`;
- допускается явное указание типа: `let x: Int = 10`.

**Операторы.** Включены арифметические (`+`, `-`, `*`, `/`, `%`), логические (`&&`, `||`, `!`), сравнительные (`==`, `!=`, `<`, `<=`, `>`, `>=`), а также присваивание (`=`) и составное присваивание (`+=`, `-=`).

**Условные операторы.** В подмножество входят `if...else` и `switch...case` с веткой `default`.

**Циклы.** Используются:
- `for-in` для перебора диапазонов и массивов;
- `repeat...while` как эквивалент `do...while`.

**Функции.** Поддерживаются:
- объявление функций с параметрами и возвращаемым значением;
- рекурсивные функции;
- локальные функции не включаются в первую версию интерпретатора.

**Структуры данных.** Включены:
- массивы `Array<T>` в синтаксисе `[]`;
- словари `Dictionary<K, V>` в синтаксисе `[:]`;
- кортежи `()` для простых составных значений.

**Пользовательские типы.** Включены простые структуры `struct` с полями и методами без наследования.

Ограничения подмножества: отсутствуют классы, протоколы, обобщения, замыкания, опциональные типы и обработка ошибок `try/catch`.

### 3 ОПРЕДЕЛЕНИЕ ИНСТРУМЕНТАЛЬНОЙ ЯЗЫКОВОЙ СРЕДЫ

**Язык реализации интерпретатора:** F# на .NET SDK 9.0.310. F# является функциональным языком, хорошо подходящим для разработки интерпретаторов благодаря выражениям сопоставления с образцом и неизменяемым структурам данных.

**Язык моделирования:** Swift (подмножество языка). В контейнере установлены системные зависимости, необходимые для сборки Swift, однако сам инструмент `swift`/`swiftc` не обнаружен, поэтому версия Swift будет уточнена после установки toolchain.

**Операционная система:** Debian GNU/Linux 12 (bookworm).

**Компьютер:** PC.

**Редактор кода:** Visual Studio Code.

**Листинг 3 – файл Example3.swift.**
```swift
import Foundation

struct Point {
    var x: Int
    var y: Int

    func distanceSquared() -> Int {
        return x * x + y * y
    }
}

func factorial(_ n: Int) -> Int {
    if n <= 1 {
        return 1
    }
    return n * factorial(n - 1)
}

let p = Point(x: 3, y: 4)
let d2 = p.distanceSquared()
let fact = factorial(5)

print("d2:", d2)
print("fact:", fact)
```
